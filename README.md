# PreciseAds — Precision Targeting for Edtech

A working prototype built for the MBA Major Project titled
"Precise Targeting Using AI Automation for the Edtech Industry,"
submitted to the Department of Management Studies, IIT Delhi.

**Live demo:** https://riddhi-mazumdar.github.io/precise-ads-mba-major-project/

## What this is

A lightweight, DPDP-compliant dashboard that reads website visitor
behaviour, classifies visitors into five intent categories, matches
those categories back to ad campaigns, and produces rupee-denominated
targeting recommendations a marketer can act on the same day.

Four modules: Money Leak Detector, Budget Allocator, Competitor
Intelligence, Weekly Tracker.
